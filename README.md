# Учебная практика по УП04
Разработка аналогов команд линукс с помощьюязыка GO
# Как использовать ls
Требуется запустить файл ls
```
./ls
```
# Поддерживаемые флаги для ls
	-R - рекурсивный вывод
	-r - обратный порядок 
	-a - включить скрытые файлы в вывод
    -l - длинный формат
	-h - читаемый размер
